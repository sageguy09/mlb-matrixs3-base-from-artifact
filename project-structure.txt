├── assets/                    # Static project assets
│   ├── fonts/                 # Font files for displays

│
├── host_scripts/              # Scripts that run on the host computer

│

## Changes from Original MLB Scoreboard Structure

### File Moves/Reorganization:
- Renamed `packages/display/matrix.py` → `packages/ui/display.py`
- Moved `packages/utils/network.py` → `packages/network/wifi.py`
- Kept `packages/mlb_api/` directory structure
- Added `packages/sensors/` directory for system monitoring
- Retained `packages/utils/` for logger and time utilities

### New Files:
- `boot.py` - Initial configuration that runs at startup
- `debug.py` - Debugging utilities
- `home_menu.py` - Menu system for navigation
- `lib/README.txt` - Library installation instructions
- `assets/fonts/README.txt` - Font information
- `packages/sensors/system.py` - System monitoring
- `tests/test_network.py` - Network connectivity tests

### Updates to Existing Files:
- `code.py` - Updated to work with new package structure
- `.gitignore` - Enhanced with project-specific patterns
- `settings.toml` - Configure MLB scoreboard with new structure

### Retained Files:
- All MLB API functionality
- Screen implementation logic
- Team color definitions
- All animation and layout code
- All deployment and host scripts
